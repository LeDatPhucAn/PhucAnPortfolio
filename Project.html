Automated Facebook Messaging

With the help of chatGPT, I developed a Python-based automation script that uses Selenium WebDriver to send preset messages on Facebook Messenger, simulating human interactions. This tool is particularly useful for automating outreach or handling inquiries on various Facebook pages. It supports a diverse range of predefined messages and mimics human behavior to reduce detection as a bot.

Key Features:

  • Automated Messaging: The bot sends predefined messages across a list of Facebook pages. Each page receives a random version of the messages from a set of templates, which are customizable.
  • Human-like Interaction: To avoid bot detection, the script simulates realistic human behavior with randomized typing speeds, mouse movements, and scrolling. It even includes the ability to simulate mistakes or typos.
  • Randomized Text Versions: Multiple message versions are prepared, ensuring each interaction is slightly different, adding to the authenticity of the communication.
  • Logging System: The bot logs both successful and failed messaging attempts, allowing for review and potential follow-up. It also remembers previously messaged pages, preventing duplicate interactions in future runs.
  • Message Extraction: It uses XPath to locate message bubbles in the chat window and extract the text content of the messages.
  • Data Storage: It collects all responses from different pages and saves them in a structured JSON file (responses.json) for easy access.


    • Use cases:

  • Automating outreach on multiple Facebook pages.
  • Automating responses retrieval on multiple Facebook pages.
  • Conducting bulk messaging for promotional purposes or information gathering through inquiries.
